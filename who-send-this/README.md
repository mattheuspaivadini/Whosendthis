## About

**Who Send This?** is a minimal, fully anonymous messaging web app. Anyone can send a message without revealing who they are, and anyone can receive a random anonymous message posted by someone else.

The project is built for those who value **ephemeral, anonymous expression** — confessions, thoughts, compliments, or anything else — without fear of judgment or repercussion.

### How it works

- **Send a message** — Type up to 500 characters and submit. No identity, no IP, no cookies are stored.
- **Receive a random message** — Get a random message from the pool of anonymous posts. You can keep fetching new ones.

No accounts. No logins. No history.
